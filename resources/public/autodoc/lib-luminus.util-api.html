cache

    macro

    
Usage: (cache id content)


    
checks if there is a cached copy of the content available,
if so the cached version is returned, otherwise the content
is evaluated

    
    
    
  

    

    

    
format-time

    function

    
Usage: (format-time time)
       (format-time time fmt)


    
formats the time using SimpleDateFormat, the default format is
"dd MMM, yyyy" and a custom one can be passed in as the second argument
